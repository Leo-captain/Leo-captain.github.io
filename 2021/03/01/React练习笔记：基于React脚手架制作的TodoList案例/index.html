八、动态初始化列表

十一、在Item组件上绑定鼠标的移入移出效果。移入和移出均使用同一个事件处理函数handleMouse()，移入传入参数true，移出传入参数false。由于需要向handleMouse传参，因此需要将该函数写成高阶函数，通过闭包的形式返回一个函数供鼠标在移入移出时调用。

给Item组件绑定一个状态mouse，在handleMouse返回的函数中调用setState方法来修改mouse的值，鼠标进入时为true，鼠标离开时为false。通过状态mouse的改变驱动Item组件的渲染。
